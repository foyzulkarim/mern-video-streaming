## Installation and run the projects

-   Install server side project and start server

```sh
$ cd server
$ npm install
$ npm start
```

-   Install client side project and start client

```sh
$ npm i -g yarn
$ cd client
$ yarn install
$ yarn start
```

View the website at: http://localhost:3000
